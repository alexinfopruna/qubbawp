== Description ==
Official lightGallery WordPress plugin. Create stunning image and video galleries for your WordPress website in minutes. Responsive, touch-friendly, and easy to use.

### Features
- Fully responsive.
- Drag and drop gallery builder
- Virtual slides
- Highly optimized for touch devices.
- Mouse drag supports for desktops.
- String translation
- Justified gallery layout support
- Supports WordPress default gallery
- Supports any kind of HTML markup
- Add lightBox suppor for all images on the page
- 20+ Hardware-Accelerated CSS3 transitions.
- Dynamic mode.
- Inline gallery
- Responsive images.
- Multiple galleries on one page.
- Easily customizable via CSS (SCSS) and Settings.
- Smart image preloading and code optimization.
- Keyboard Navigation for desktop.
- SVG icons.
- Accessibility support.
- Mousewheel support
- HTML captions
- Swipe down/up to close the gallery
And many more.

== Installation ==

= In WordPress =

* Go in your Dashboard to \'Plugins\' -> \'Install\'
* Search for \'LightGallery WPp\'
* Click on \'Install\'
* Activate the plugin through the \'Plugins\' menu in WordPress
* Now you can go to the Settings-Page from the Admin Menu

= Manually =

* Download the Plugin
* Go in your Dashboard to \'Plugins\' -> \'Install\'
* Click on Upload
* Click on \'Choose file\' and choose the downloaded Zip-file
* Click on \'Install\'
* Activate the plugin through the \'Plugins\' menu in WordPress
* Now you can go to the Settings-Page from the Admin Menu

== Frequently Asked Questions ==

= Does lightGallery support responsive layouts? =
Yes, lightGallery is 100% responsive and highly optimized for mobile and touch devices. lightGallery supports swipe/drag to navigate different slides and swipe down to close the gallery? 

= Can I replace my existing WordPress gallery with lightGallery? =
Yes, by default lightGallery adds lightbox support to your default WordPress galleries. It can be turned off or customized from the plugin settings interface. 

= Can I add lightbox support for my individual images? =
Yes, if your image is linked to a bigger picture, lightGallery automatically displays it when the user clicks on the image. Your users can view the full-sized photos without leaving your website. And, of course, this can be turned off or customized from settings if you want to. 

= Does lightGallery support custom HTML markup? =
Yes, lightGallery supports almost any kind of HTML markups. It is easy to add lightbox gallery support for your existing thumbnails. 

= Do I need to know coding to use lightGallery? =
Absolutely not. You can simply go to the plugin settings interface, select photos you want to display in the gallery, re-arrange it however you like using our drag and drop builder. That's it. You can create and customize your beautiful photo gallery without writing a single line of code.
